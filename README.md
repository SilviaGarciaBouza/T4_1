# T4_1 - Bar Order App

Este proyecto consiste en una aplicación multiplataforma para gestionar los pedidos de un bar, creada con Flutter. Permite anotar qué pide cada mesa, visualizar los pedidos y organizar las comandas de forma rápida.

---

## Características
Las acciones que puedes hacer con esta aplicación son:
- **Ver la lista de pedidos**: Ves todas las mesas que están atendiendo, cuántos productos tienen y el total a pagar.
- **Tomar nota**: Puedes crear un nuevo pedido escribiendo el número de mesa.
- **Elegir productos**: Tienes una lista con cafés, zumos y tés, para ir añadiendo al pedido.
- **Resumen**: Antes de cerrar, puedes ver un desglose de lo que se ha pedido.

---

## Tecnología
- **Arquitectura:** MVVM (Model-View-ViewModel).
- **Lenguaje:** Dart.
- **Framework:** Flutter.

---

## Flujo de Navegación
El proyecto utiliza dos tipos de navegación:
- Navegación **Imperativa** (`Navigator.push`).
- Navegación con **Ruta con Nombre** (`Navigator.pushNamed`).

## Cómo usar la aplicación
1.  Abre la terminal en la carpeta principal.
2.  Ejecuta `flutter pub get`.
3.  Ejecuta `flutter run`.
