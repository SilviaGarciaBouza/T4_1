# T4_1 - Bar Order App

Este proyecto consiste en una aplicación multiplataforma para gestionar los pedidos de un bar, creada con Flutter. Permite anotar qué pide cada mesa, ver el total y organizar las comandas de forma rápida.

## Características

- **Lista de pedidos**: Ves todas las mesas que están atendiendo, cuántos productos llevan y el total a pagar.
- **Tomar nota**: Puedes: crear un nuevo pedido escribiendo el número de mesa.
- **Elegir productos**: Tienes una lista con cafés, zumos, tés, etc., para ir sumando al pedido.
- **Resumen**: Antes de cerrar, puedes ver un desglose de lo que se ha pedido.


## Tecnología

- **Arquitectura:** MVVM (Model-View-ViewModel).
- **Lenguaje:** Dart.
- **Framework:** Flutter.

## Flujo de Navegación

El proyecto utiliza dos tipos de navegación:

- Navegación **Imperativa** (`Navigator.push`).
- Navegación con **Ruta con Nombre** (`Navigator.pushNamed`).

## Cómo usarla

1.  Abre la terminal en la carpeta principal.
2.  Ejecuta `flutter pub get`.
3.  Ejecuta `flutter run`.
